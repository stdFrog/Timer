간단한 타이머 프로그램입니다.
시스템과 관련된 설정은 일절 건드리지 않습니다.

API 다루면서 처음 만들었던게 타이머였는데 이번에 다시 새로 만들었습니다.
시,분,초 등은 최대 5자리 수까지만 입력을 받도록 설정해뒀습니다.
가끔 1200초, 3600초, 8400초, 120분 등의 변환하지 않은 시간을 그대로 사용할 때가 있어서 굳이 예외 처리하지 않았습니다.

프로그램을 실행하면 기본적으로 25분의 타이머가 설정되어 있습니다.

총 3개의 버튼이 배치되어 있으며 각각은 이름만으로도 어떠한 기능을 하는지 직관적으로 알 수 있으므로 따로 설명을 추가하진 않겠습니다.
외에 별 다른 기능은 없으며 설정한 시간이 지나 타이머가 종료되면 TIME OUT 문장과 비프음이 출력됩니다.
